# Test1 - Python 学习项目

这是一个用于学习 Python 的项目，包含了各种示例代码和练习。

## 项目结构

```
test1/
├── README.md              # 项目说明
├── requirements.txt       # 依赖包列表
├── .gitignore            # Git 忽略文件
├── main.py               # 主程序入口
├── examples/             # 示例代码
│   ├── basics/           # 基础语法
│   ├── oop/              # 面向对象编程
│   ├── modules/          # 模块和包
│   └── advanced/         # 高级特性
├── exercises/            # 练习题
├── data/                 # 示例数据
└── notes/                # 学习笔记
```

## 安装和使用

1. 创建虚拟环境：
```bash
python -m venv venv
```

2. 激活虚拟环境：
```bash
# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

3. 安装依赖：
```bash
pip install -r requirements.txt
```

4. 运行主程序：
```bash
python main.py
```

## 学习建议

1. 从 `examples/basics/` 开始，逐步学习基础语法
2. 完成 `exercises/` 中的练习题
3. 阅读代码注释，理解每个示例的作用
4. 尝试修改示例代码，加深理解
